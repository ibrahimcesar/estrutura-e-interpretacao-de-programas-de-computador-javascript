## Visão Geral

Esta seção final mostra como integrar código compilado e interpretado:

- Interpretação e compilação no mesmo sistema
- Chamadas entre código compilado e interpretado
- Ambiente compartilhado
- Primitivos compartilhados
- Sistema híbrido de execução

## Tópicos Principais

- Interface compilador-interpretador
- Funções compiladas chamando funções interpretadas
- Funções interpretadas chamando funções compiladas
- Ambiente global compartilhado
- Operações primitivas compartilhadas

## Sistema Híbrido

- Registro de funções compiladas no ambiente
- Detecção de tipo de função (compilada vs. interpretada)
- Estratégias de aplicação diferentes
- Vantagens de sistema híbrido

## Implementação

- Modificações no avaliador para suportar código compilado
- Modificações no compilador para interagir com interpretador
- Entrada e saída do código compilado
- Driver loop unificado
