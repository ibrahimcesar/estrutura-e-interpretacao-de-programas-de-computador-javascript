## Visão Geral

Esta seção fundamental explica a coleta de lixo (garbage collection):

- Conceito de garbage collection
- Algoritmo stop-and-copy
- Identificação de memória acessível
- Compactação de memória
- Recuperação de células não utilizadas

## Tópicos Principais

- Garbage collection (coleta de lixo)
- Algoritmo stop-and-copy
- Espaços de memória: working memory e free memory
- Raízes de garbage collection
- Varredura de estruturas acessíveis
- Compactação e relocação
- Análise de complexidade

## Algoritmo Stop-and-Copy

- Divisão de memória em dois espaços
- Cópia de células acessíveis
- Atualização de ponteiros
- Troca de espaços
- Broken-heart cells
