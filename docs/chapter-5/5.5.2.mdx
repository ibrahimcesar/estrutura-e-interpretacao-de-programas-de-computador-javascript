## Visão Geral

Esta seção detalha a compilação de diferentes tipos de expressões:

- Compilação de expressões literais e nomes
- Compilação de expressões lambda
- Compilação de condicionais
- Compilação de atribuições e declarações
- Geração de código para cada tipo de expressão

## Tópicos Principais

- Literais e constantes
- Lookup de nomes (variáveis)
- Expressões lambda e closures
- Condicionais (if)
- Atribuições
- Declarações (const, let)
- Sequências
- Retornos

## Geração de Código

- Código para literais (constantes)
- Código para acesso a variáveis
- Código para criar funções (closures)
- Código para avaliação condicional
- Código para modificação de ambientes
