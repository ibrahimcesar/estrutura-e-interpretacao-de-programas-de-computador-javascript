# Sobre a Tradução Brasileira

Esta tradução para português brasileiro do SICP JS é um projeto de código aberto coordenado por Ibrahim Cesar, com o objetivo de tornar este conhecimento fundamental acessível para a comunidade de língua portuguesa.

## Ferramentas e Tecnologia

Este projeto de tradução conta com a assistência do **Claude Code**, uma ferramenta de IA desenvolvida pela Anthropic que auxilia no processo de tradução e manutenção da documentação.
